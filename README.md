# Cybersecurity Regulatory Mapping

## Overview
A cybersecurity regulatory mapping document mapping 16 organisational assets and controls to PCI DSS, ISO 27001 and NIST CSF — built to complement the Cybersecurity Risk Register project.

## What's Included
- 16 assets mapped across three major frameworks
- Compliance status for each asset
- Gap analysis and remediation actions
- Priority and target dates for each gap
- Legend and summary tabs

## Frameworks Referenced
- PCI DSS — Payment Card Industry Data Security Standard
- ISO 27001 — Information Security Management System
- NIST CSF — NIST Cybersecurity Framework

## How It Connects to the Risk Register
Each asset in this document directly corresponds to an asset in the Cybersecurity Risk Register — the controls, threats and owners are consistent across both projects demonstrating an end to end GRC approach.

## Skills Demonstrated
- Regulatory mapping and compliance assessment
- Gap analysis
- Framework knowledge — PCI DSS, ISO 27001, NIST CSF
- GRC documentation
- Risk and compliance alignment
